Comandos Cisco IOS

Para configurar um dispositivo com IOS da CISCO, exitem dois níveis de execução. Usuário comum e privilegiado

  1. Comandos Modo Usuário
    • enable
    • disable
    • exit

  2. Comandos exec privilegiado
    • configure terminal
        • Entra no modo de configuração global

    • line console 0
        • Configuração do entrada console

    • line vty 0 15
        • Configuração de acesso por ssh ou telnet

    • interface fastethernet 0/1
        • Configuração da interface FE

    • interface gigabitethernet 0/1
        • Configuração da interface gb

    • exit
        • Volta um nível a baixo na hierarquia

    • end ou crtl + z
        • Retorna para o modo EXEC priv
